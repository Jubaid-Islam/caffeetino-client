# Caffetino Client

Frontend of the Caffetino coffee web store, built to deliver a smooth shopping experience with secure authentication, online payments, and intuitive order management.

## Features

- Responsive coffee store with product browsing, category filtering, and search functionality.
- Secure authentication using Firebase Authentication and JWT.
- Shopping cart, wishlist, and checkout with Stripe payment integration.
- User dashboard for managing profile, orders, and account information.
- Dynamic product management with real-time stock updates.
- Modern UI with smooth animations and a mobile-friendly design.

## Tech Stack

- React.js
- React Router
- Tailwind CSS
- DaisyUI
- Firebase Authentication
- Axios
- Stripe
- React Hook Form
- SweetAlert2
- React Hot Toast
  



## Core Modules

- Authentication
